Os elementos h2 e h3 são usados para definir títulos e subtítulos da principais seções de uma página web. O h1 é como se fosse o título, os elementos h2 são os capítulos e os elementos h3 são seções dos capítulos. Utilize somente um h1 por página, já o h2 e h3 pode ser utilizados bem mais vezes.
